# 参考资料

> 学习 `just-go` 推荐配套的外部资料。优先官方资源，其次社区公认精品，最后中文书单。

---

## 一、官方资源

- [Go 官网（go.dev）](https://go.dev/)
- [Tour of Go](https://go.dev/tour/) —— 浏览器内的交互式入门教程
- [Effective Go](https://go.dev/doc/effective_go) —— Go 风格指南，必读
- [Go Modules Reference](https://go.dev/ref/mod) —— Go module 的权威文档
- [Go Memory Model](https://go.dev/ref/mem) —— 并发理解的根
- [Go Standard Library (pkg.go.dev/std)](https://pkg.go.dev/std)
- [Go Blog](https://go.dev/blog/) —— 官方博客，新特性首发地

## 二、中文书单

- 《Go 程序设计语言》—— Alan A. A. Donovan / Brian W. Kernighan（也称"Go 圣经"）
- 《Go 语言高级编程》—— 柴树杉 / 曹春晖（深入工程化）
- 《Go 语言项目开发实战》—— 孔令飞
- 《Go 专家编程》—— 任洪彩
- 《架构整洁之道》—— Robert C. Martin（与阶段三第 12 章配套）
- 《领域驱动设计》—— Eric Evans（与阶段三第 13 章配套）
- 《微服务架构设计模式》—— Chris Richardson（与阶段三第 14 章配套）
- 《数据密集型应用系统设计》—— Martin Kleppmann（与阶段三第 15 章配套）

- [Go 夜读](https://reading.developerlearning.cn/) —— 中文社区
