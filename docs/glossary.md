# 术语表

> 本文件由各章节增量追加：每个 `chapter-NN-*` change 在引入新术语时 SHALL 追加一行到本表，并在"出现章节"列注明章节编号。
>
> 排序规则：按术语英文名字母序。

| 术语 | 英文 | 解释 | 出现章节 |
| ---- | ---- | ---- | ------- |
| 字节 | byte | `uint8` 的别名，表示单个字节；字符串可按 byte 遍历 | 02 |
| 闭包 | closure | 函数值引用其定义外部的变量，多次调用间可共享状态 | 02 |
| 延迟调用 | defer | 函数返回前按后进先出顺序执行 deferred 调用，常用于资源释放 | 02 |
| 模块代理 | GOPROXY | 控制 `go` 命令从何处下载模块的环境变量，国内常设为镜像以加速依赖拉取 | 01 |
| 导入路径 | import path | 唯一标识一个包的字符串，由 `go.mod` 的模块前缀拼上包所在目录路径而成 | 01 |
| 常量计数器 | iota | 在 `const (...)` 块中从 0 递增的隐式常量生成器 | 02 |
| 模块 | module | 一组同属一个版本单元的 Go 包，根目录含 `go.mod`，其 `module` 行定义模块路径前缀 | 01 |
| 命名返回值 | named result | 在函数签名中为返回值命名，便于 `defer` 修改最终返回结果 | 02 |
| 包 | package | Go 代码组织与复用的基本单位，同一目录下文件共享同一 `package` 名 | 01 |
| Unicode 码点 | rune | `int32` 的别名，表示一个 Unicode 字符 | 02 |
| 可变参数 | variadic parameter | 函数参数写成 `...T`，调用时可传入零个或多个 T 类型实参 | 02 |
| 可见性 | visibility (exported/unexported) | 由标识符首字母大小写决定：大写为导出（跨包可访问），小写为未导出（仅包内可见） | 01 |
| 零值 | zero value | 变量声明后未显式赋值时的默认值（如 `int` 为 0，`string` 为 `""`） | 02 |
