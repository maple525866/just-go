# 02. 语法基础

> 阶段：① 语法精通 ｜ 难度：⭐⭐☆☆☆ ｜ 预计耗时：1 天

## 🎯 学习目标

掌握变量、常量、基本类型、控制流、函数、包的全部基础语法。

## 🧩 关键知识点

- 变量声明（`var` / `:=`）、零值、常量与 `iota`
- 基本类型（数值 / 布尔 / 字符串 / `rune` / `byte`）与类型转换
- 控制流（`if` / `for` / `switch` / `defer`）
- 函数：多返回值、命名返回、可变参数、闭包
- 包的组织、可见性（首字母大小写）
