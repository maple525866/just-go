# 01. Hello, Go

> 阶段：① 语法精通 ｜ 难度：⭐☆☆☆☆ ｜ 预计耗时：3 小时

## 🎯 学习目标

装好 Go 工具链，跑通第一个程序，理解 `go run / build / mod` 三件套与项目结构。

## 🧩 关键知识点

- Go 安装与环境变量（GOPATH / GOROOT / GOPROXY）
- `go mod init / tidy / run / build / test` 命令
- 包（package）、import 路径、`main` 函数
- VS Code / GoLand 基础配置与调试器
